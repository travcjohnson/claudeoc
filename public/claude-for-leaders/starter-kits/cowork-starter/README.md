# Claude Cowork starter base

A healthy folder to start a **Cowork project** on the right foot. Cowork works
on your local files and carries a task all the way to the deliverable. This folder
gives it the **context** and the **rules** to do it well, from the very first time.

## Import this folder into Cowork (3 steps)

1. Open **Claude Desktop → Cowork → Projects → Create your first project**.
2. Choose **"Use an existing folder"** and point to **this folder**.
3. In the project's **Instructions** field, paste the contents of `INSTRUCTIONS.txt`.

That's it. Cowork will read the context and keep its memory from one task to the next.

## What the folder contains

```
cowork-starter/
├── README.md               ← this file
├── INSTRUCTIONS.txt        ← to paste into the project's Instructions field
├── context/
│   └── HOW-I-WORK.md       ← who you are, your business, your preferences
├── system/
│   └── SYSTEM-RULES.md     ← rules, tone, format, guardrails
├── memory.md               ← what Cowork must remember from one task to the next
├── plan.md                 ← the framing: objective and steps
├── todo.md                 ← the state of the moment
└── outputs/                ← where Cowork puts the deliverables
```

## Recommended reading order for Cowork
1. `context/HOW-I-WORK.md`
2. `system/SYSTEM-RULES.md`
3. `plan.md`, then the task at hand.
